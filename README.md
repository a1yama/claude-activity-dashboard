# claude-activity-dashboard

Claude Code の活動ログをブラウザで閲覧・分析するダッシュボード。

## セットアップ

```bash
make setup
```

## 起動

```bash
make serve
```

http://localhost:8765 でダッシュボードが開きます。

## データ更新

- ブラウザから: `/-/refresh` にアクセスして「更新実行」ボタン
- CLIから: `make ingest`

## 主なビュー

- **日別アクティビティ**: 日ごとのセッション数・メッセージ数
- **プロジェクト別サマリー**: プロジェクトごとの利用状況
- **時間帯別分布**: 何時にClaude Codeを使っているか
- **ツール使用ランキング**: どのツールが多く使われているか
